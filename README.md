Smart City Traffic & Emergency Response AI System is an Artificial Intelligence-based traffic management solution developed using Python, Flask, HTML, and CSS. The system is designed to handle both civilian and emergency vehicle requests in a smart city environment by intelligently selecting and executing only the AI modules required for a specific request.

The project integrates multiple AI techniques, including Artificial Neural Networks (ANN), First-Order Logic Knowledge Bases, Constraint Satisfaction Problems (CSP), and Search Algorithms such as A*, BFS, and UCS.

Key features include:

- Intelligent request routing and preprocessing
- Emergency vehicle priority prediction using ANN
- Policy-based decision making using a Knowledge Base
- Traffic signal scheduling through CSP techniques
- Route optimization using AI search algorithms
- Real-time emergency response management
- Interactive web interface built with Flask

The system supports route planning, policy verification, signal control allocation, and full emergency response operations, helping improve traffic flow and emergency vehicle movement within a smart city infrastructure.

This project was developed as part of the Artificial Intelligence Lab course and demonstrates the practical application of Machine Learning, Knowledge Representation, Constraint Satisfaction, and Search Techniques in a real-world smart city scenario.
